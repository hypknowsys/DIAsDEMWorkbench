Description

Top of the Page

Given a user-specific information need, this task requests a list of relevant URLs from the Google Web service. In addition to specifying query terms, one's information need can be expressed in detail by means of several Google-specific parameters. For example, the search can be restricted to Web pages, which have been published on known Web sites, in a specific language, or within a certain date range. In particular, this task can be employed to collect a list thematically relevant URLs for subsequent download and semantic text annotation.

Prerequisites

Top of the Page

An established Internet connection is assumed in order to successfully submit the query and download relevant URLs. In addition, the user needs a personal Google API search key, which is freely available as of May 2004 for research and personal usage. To obtain a search key, the user has to set up a Google account at http://www.google.com/apis/ and declare to abide by the license terms.

Result

Top of the Page

After calling the Google Web service, the most relevant URLs are stored in a plain text file. This so-called download URL file can subsequently serve as input to the task Import HTML Files from the Web in order to import the textual file contents into a DIAsDEM collection for further processing. See below an example for a download URL file:

# Google Search Result (0.302296 sec.), Start Index: 1, End Index: 2
# Search Date: Wed May 12 17:02:42 CEST 2004
# Search String: "DIAsDEM Workbench" filetype:html OR filetype:htm 
http://www.diasdem.de/index.html
http://www.hypknowsys.org/index.html

Remarks

Top of the Page

As of May 2004, the Google Web APIs service is in beta release. All interfaces are subject to change as Google refines and extends the APIs. Please see the terms of use (in the file "LICENSE.txt") for more information.

Parameters

Top of the Page

Download URL File

Output parameter: Valid local file name of new or existing download URL file, which is either created or replaced by DIAsDEM Workbench; file extension: .url; default value: project property Search the Web for HTML Files: Default Download URL File

Google Search Key

Input parameter: String, which was assigned to the user as a personal Google search key; default value: project property Search the Web for HTML Files: Google Search Key

Google Search String

Input parameter: String, which defines the query terms in a Google-specific syntax; examples: "DIAsDEM Workbench" and Star Wars Episode +I; default value: project property Search the Web for HTML Files: Google Search String

Max. Number of Results

Input parameter: Integer greater than zero, which defines the maximum number of most relevant URLs to be retrieved from Google; default value: project property Search the Web for HTML Files: Max. Number of URLs to be Retrieved

Restrict Sites

Optional input parameter: Comma-separated list of Web sites, which restricts the search to Web pages published on the specified Web sites; examples: www.berlin.de as well as www.diasdem.de,www.hypknowsys.org; default value: project property Search the Web for HTML Files: Restrict Search to these Web Sites

Restrict File Type

Optional input parameter: Comma-separated list of file extensions, which restricts the search file of the specified type or file extension; example: html,htm,php; default value: project property Search the Web for HTML Files: Restrict Search to these File Types

Restrict Date (Start)

Optional input parameter: Start date in YYYY-MM-DD format, which indicates the begin of the date range to restrict the search to Web pages published within a specific date range; examples: 2004-01-01 for January 1, 2004, and 2004-12-31 for December 31, 2004; default value: project property Search the Web for HTML Files: Start Date that Restricts Search

Restrict Date (End)

Optional input parameter: End date in YYYY-MM-DD format, which indicates the end of the date range to restrict the search to Web pages published within a specific date range; examples: 2004-01-01 for January 1, 2004, and 2004-12-31 for December 31, 2004; default value: project property Search the Web for HTML Files: End Date that Restricts Search

Restrict Language

Optional input parameter: One Google-specific language identifier, which restricts the search to language-specific Web pages; examples: lang_en for English Web pages and lang_de for German Web pages; default value: project property Search the Web for HTML Files: Restrict Search to this Language
