Description

Top of the Page

This task finds and replaces labels of processed text units in the specified DIAsDEM collection. Use cases: Correct spelling mistakes made in a cluster label file that have been propagated into the collection by the task Tag Text Units. Prior to deriving a conceptual DTD, replace auxiliary labels such as "Ignore Topic" with "-" (i.e., the cluster label denoting qualitatively unacceptable text unit clusters) to prevent them from being included in the conceptual DTD.

Prerequisites

Top of the Page

None.

Result

Top of the Page

If the label of a processed text unit in the specified DIAsDEM collection equals Find Label, it will be replaced by Replace With. Note that both the iteration number and the cluster ID of processed text units remain unchanged.

Parameters

Top of the Page

Collection File

Input parameter: Valid local file name of existing collection file; file extension: .dcf; default value: project property Default Collection File

Find Label

Input parameter: Case-sensitive label of processed text units to find; example: Ignore Topic; default value: project property Replace Labels of Text Units: Default Find Label

Replace With

Input parameter: Label of processed text units to replace; example: -; default value: project property Replace Labels of Text Units: Default Replace With

Log File

Input parameter: Valid local file name of log file, which will either be created or replaced; log file will contain details of text units whose labels have been replaced; file extension: .txt; default value: project property Replace Labels of Text Units: Default Log File Name
