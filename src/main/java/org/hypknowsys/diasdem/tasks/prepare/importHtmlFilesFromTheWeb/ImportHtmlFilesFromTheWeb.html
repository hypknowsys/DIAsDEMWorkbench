Description

Top of the Page

If the user wants to employ DIAsDEM Workbench to semantically annotate the textual content of HTML documents, the respective files must first be downloaded from the Web and pre-processed to extract their textual content. After having searched relevant HTML documents using for example the task Search the Web for HTML Files, this task downloads HTML files, which are listed in a so-called download URL file, removes all HTML tags, and finally imports the resulting plain texts into a document collection.

Prerequisites

Top of the Page

An established Internet connection is assumed in order to successfully download HTML documents referenced in the download URL file. This file contains the URLs of HTML documents to be downloaded. Empty lines as well as comment lines starting with '#' are ignored. See below an example for a download URL file generated by the task Search the Web for HTML Files: 

# Google Search Result (0.302296 sec.), Start Index: 1, End Index: 2
# Search Date: Wed May 12 17:02:42 CEST 2004
# Search String: "DIAsDEM Workbench" filetype:html OR filetype:htm 
http://www.diasdem.de/index.html
http://www.hypknowsys.org/index.html

Result

Top of the Page

Each HTML document referenced in Download URL File is first retrieved and saved with file extension .html in Text File Directory for debugging purposes. Thereafter, each HTML file is detagged by removing HTML tags and scripts (e.g., Javascript), because markup is assumed to be useless for semantic text annotation. The resulting plain text file is stored with File Name Extension in Text File Directory. Finally, all text files are imported into the document collection specified by Collection File.

Remarks

Top of the Page

The heuristic detagging algorithm should be refined prior to employing this task in a productive environment. It should be stressed that only HTML-encoded files (i.e., as opposed to binary such as PDF files) can be pre-processed and imported by this task. Hence, users must ensure that the specified download URL file only comprises references to HTML-encoded documents.

Parameters

Top of the Page

Collection File

Input parameter: Valid local file name of existing collection file; file extension: .dcf; default value: project property Default Collection File

Download URL File

Input parameter: Valid local file name of file, which contains URLs of all HTML documents to be downloaded, pre-processed, and imported; file extension: .url; default value: project property Import HTML Files from the Web: Default Download URL File

Text File Directory

Output parameter: Existing local directory, which will contain downloaded HTML files as well as detagged text files to be imported; default value: project property Default Text File Directory

File Name Extension

Input parameter: Names of created text files (i.e., detagged HTML documents) will have this file name extension; default value: .txt
